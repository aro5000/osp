# OSP

**O**bnoxious

**S**lack

**P**erson

![](docs/osp.gif)

## Build
First, [get Zig](https://ziglang.org/download/) (`brew install zig` on Mac)

```
zig build
mv zig-out/bin/osp /somewhere/in/your/path
```
